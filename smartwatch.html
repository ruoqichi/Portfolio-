Description

IAT 235 is a second-year program that introduces the theory and practice of designing visual representations of information. The third individual project is to design five original smartwatch interface; 3 common concepts and 2 completely original ones.

1 / 5
Allows user to control their Smart Home device. (Non-standard)
2 / 5
Allows user to see dogs around them, with the permission of their owners. (Non-standard)
3 / 5
Allows user to listen to music. (Standard)
4 / 5
Allows user to track their daily steps. (Standard)
5 / 5
Allows user create a To-Do List. (Standard)
